# Smart Waste Segregator using Arduino

A smart waste segregation system developed using Arduino UNO, IR sensor, moisture sensor, and servo motor. The system automatically detects waste and separates it into wet and dry categories.

---

## Features

- Automatic waste detection
- Wet and dry waste segregation
- Servo motor automation
- IR sensor based object detection
- Compact prototype model

---

## Components Used

- Arduino UNO
- IR Sensor
- Moisture Sensor
- Servo Motor
- Jumper Wires
- Power Supply

---

## Working

1. IR sensor detects the presence of waste.
2. Moisture sensor identifies whether the waste is wet or dry.
3. Arduino processes the sensor data.
4. Servo motor rotates accordingly to separate the waste.

---

## Circuit Connections

| Component | Arduino Pin |
|----------|-------------|
| IR Sensor | Pin 2 |
| Moisture Sensor D0 | Pin 4 |
| Moisture Sensor A0 | A0 |
| Servo Motor Signal | Pin 9 |
